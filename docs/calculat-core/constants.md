# 常量定义

> `src/core/constants.ts` - 身体部位基准、参照物尺寸和级别定义

---

## 📋 概览

常量模块定义了计算所需的所有基准数据：

| 常量 | 说明 |
|------|------|
| `BASE_HEIGHT` | 基准身高（1.75m） |
| `BASE_BODY_PARTS` | 身体部位基准尺寸 |
| `REFERENCE_OBJECTS` | 参照物尺寸（日常物品到天体） |
| `INSECT_REFERENCES` | 昆虫参照（用于小人视角） |
| `SIZE_LEVELS` | 巨大化级别定义 |
| `TINY_LEVELS` | 缩小化级别定义 |

---

## 🧍 BASE_HEIGHT

基准身高，所有身体部位数据都基于此身高。

```typescript
export const BASE_HEIGHT = 1.75; // 米
```

### 缩放规则

不同类型的部位使用不同的缩放规则：

| 类型 | 缩放方式 | 公式 | 示例 |
|------|----------|------|------|
| 长度 | 线性 | `基准值 × 倍率` | 足长、身高 |
| 面积 | 平方 | `基准值 × 倍率²` | 指尖面积 |
| 体积 | 立方 | `基准值 × 倍率³` | 阴道容积、子宫容积 |
| 重量 | 立方 | `基准值 × 倍率³` | 单乳重量、臀部重量 |

## 📉 TINY_LEVELS

缩小化级别定义，用于判断小人的级别。

### 数据结构

```typescript
interface TinyLevel {
  name: string;        // 级别名称
  scale: number;       // 代表倍率
  description: string; // 级别描述
}

export const TINY_LEVELS: TinyLevel[] = [...];
```

### 级别列表

| 级别名称 | 代表倍率 | 描述 | 身高示例（原1.70m） |
|----------|----------|------|--------------------|
| 十分之一 | 0.1 | 十几厘米，如宠物 | 17cm |
| 百分之一 | 0.01 | 不到两厘米，如虫子 | 1.7cm |
| 千分之一_毫米级 | 0.001 | 毫米级，蚂蚁如猛兽 | 1.7mm |
| 万分之一 | 0.0001 | 肉眼难见 | 0.17mm |
| 微米级 | 0.000001 | 细胞尺度 | 1.7μm |
| 纳米级 | 0.000000001 | 分子尺度 | 1.7nm |
| 皮米级 | 0.000000000001 | 原子尺度 | 1.7pm |

### 级别判定逻辑

```typescript
// 小人级别判定使用范围匹配
for (const level of TINY_LEVELS) {
  if (scale <= level.scale * 10 && scale > level.scale / 10) {
    return { ...level, type: 'tiny' };
  }
}
```

## 💡 使用示例

### 访问身体部位数据

```typescript
import { BASE_BODY_PARTS, BASE_HEIGHT } from '@/core';

// 获取原始基准值
const baseFoot = BASE_BODY_PARTS.足长; // 0.25 米

// 根据自定义原身高调整基准
const originalHeight = 1.65;
const adjustedFoot = baseFoot * (originalHeight / BASE_HEIGHT);
// 0.25 * (1.65 / 1.75) = 0.236 米
```

### 查找参照物

```typescript
import { REFERENCE_OBJECTS } from '@/core';

// 获取特定参照物尺寸
const carLength = REFERENCE_OBJECTS.轿车长度; // 4.5 米
const earthDiameter = REFERENCE_OBJECTS.地球直径; // 12,742,000 米

// 遍历所有参照物
for (const [name, size] of Object.entries(REFERENCE_OBJECTS)) {
  console.log(`${name}: ${size} 米`);
}
```

### 判断级别

```typescript
import { SIZE_LEVELS, TINY_LEVELS } from '@/core';

// 查找匹配的巨大化级别
const scale = 150; // 150 倍
const level = SIZE_LEVELS.find(
  l => scale >= l.minScale && scale < l.maxScale
);
console.log(level?.name); // "Kilo级"

// 查找匹配的缩小化级别
const tinyScale = 0.005;
const tinyLevel = TINY_LEVELS.find(
  l => tinyScale <= l.scale * 10 && tinyScale > l.scale / 10
);
console.log(tinyLevel?.name); // "千分之一_毫米级"
```
